# Task Templates

Depth-first templates for common Engram implementation tasks.

## Available Templates

- `new-mcp-tool.md` — Add a new MCP tool
- `convex-table.md` — Add/extend a Convex table and related API
- `cron-job.md` — Add a Convex scheduled job
- `lifecycle-hook.md` — Add a Claude hook safely
- `feature-implementation.md` — End-to-end feature workflow

## Usage

1. Pick the closest template.
2. Instantiate placeholders (`<feature>`, `<table>`, etc.).
3. Execute leaf tasks first.
4. Validate with `scripts/validate-golden-principles.ts`.
